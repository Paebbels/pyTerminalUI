pyTerminalUI Documentation
##########################

A set of helpers to implement a text user interface (TUI) in a terminal.


Installation/Updates
********************

Installation using PIP
======================

.. code-block:: bash

   pip3 install pyTerminalUI


Updating using PIP
==================

.. code-block:: bash

   pip3 install -U pyTerminalUI

************
